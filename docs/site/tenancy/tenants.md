# Tenants

## Description
Tenants are entities which will be using the mock server for the provided feature.
Tenants are the onboarded on the application using tenant attributes such as :

| Attribute   | Type   | Description                                                                                              |
|-------------|--------|----------------------------------------------------------------------------------------------------------|
| name        | string | This is a <b>mandatory</b> field.<br/>This is the identifying name of the application                    |
| email       | string | This is a <b>non mandatory</b> field. <br/> This is to identify the email address of tenant              |
| environment | string | This is a <b>mandatory</b> field. This is used to define in which environment the application is running |
| status      | enum   | This is a <b>mandatory</b> field.<br/> This defines in state of the tenant.                              |
